# APT-Persistence 研究作業フォルダ

この作業フォルダは、APT-Persistence を用いた正常行動復元研究の資料と分析データを整理したもの。

## まず見る場所

- 研究ドキュメント: `docs/`
- 分析に使ったCSV: `analysis_data/`
- 元データセット: `apt-persistence/`

## 推奨の読み方

最初に `docs/00_研究資料目次.md` を開く。

研究資料は `docs/` に番号付きで並べているため、基本的には `01` から順番に読めば流れが追える。

## 注意

OneDrive 管理下で一部ファイルの移動が拒否されたため、ルート直下に古い配置のファイルが残っている場合がある。

以後の作業では、整理済みの `docs/` と `analysis_data/` を正として扱う。
